# IoT World Engine

## Overview

This project simulates a smart campus environment using Python asyncio.

## Features

* High-concurrency simulation
* MQTT telemetry
* Heartbeat monitoring
* SQLite persistence

## Wokwi Hardware Prototype (POC)

* Live Simulation: https://wokwi.com/projects/459235328819945473

This prototype demonstrates a reference IoT room using ESP32, DHT22, and PIR sensors. It publishes telemetry and heartbeat messages to an MQTT broker.

## Docker

The project is containerized using Docker.
